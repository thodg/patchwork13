Monday 3rd, April 2006       Subversion !

We moved from CVS to Subversion today.
DO NOT USE CVS ANYMORE, USE SUBVERSION !

You can browse the repository online.

You will only need to checkout once with
 svn checkout https://svn.sourceforge.net/svnroot/patchwork13/

After that, the usual work cycle is to :

update your local copy with 
 svn update

modify sources

add files with 
 svn add file1 file2 ...

remove files with 
 svn rm file1 file2 ...

copy file with 
 svn cp src dest

move file with 
 svn mv src dest

check the files status with 
 svn status

if there are conflicts, either someone fixed them and you must 
 svn update

or if you fixed them you must do
 svn resolved file1 file2 ...

finally, commit your changes using 
 svn commit

Voila !

Please only commit your changes when everything COMPILES (and works if you're only upgrading stuff).

billitch
